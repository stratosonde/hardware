# Schematics v1.0

Version 1.0 hardware schematics for Stratosonde.

## Files

Place schematic PDF files in this directory.

Expected files:
- Main board schematic
- Power management schematic
- Communication schematic
- Any supporting documentation

## Naming Convention

Use descriptive filenames with version numbers:
- `stratosonde_v1.0_main_schematic.pdf`
- `stratosonde_v1.0_power_schematic.pdf`
- etc.
